Analysis of Olympic Data with GDP

Github Link : Github

~ 10 min read Fall 2021

Summary
This project was done for the class Programming for Data Science (DS5100). In the github repository info there are many componenets that were utilized for this project. The main componenets are broken down as such.

  1. Data Pull
  2. Analysis of Results
  3. Presentation of Results

Data Pull Cleaning / Processing
The data utilized for this project was all scrapped from public online sources. The Beautiful Soup package was used as well as other packages such as Pandas, Numpy, Pbar, Requests, ..etc.
Three major data sources were scrapped for this project.

  1. Olympic Details
  2. Recent GDP
  3. Historical GDP

Upon scrapping these three datasets data cleaning and operations were done for all the datasets to generate one dataframe (csv) for further analysis. All datasets collected can be found in the data folder of the repo for this project.

The methods / code for the data cleaning / processing can be found in the Cleaner class found in cleaner.py

Analysis of Results
Before running any model preliminary exploratory data analysis was conducted with the cleaned dataset. A full report of EDA can be found in the EDA Notebook.

First a correlation plot was created for the numerical values of the dataset.
Some of the main columns for the dataset are Gold, Silver, Bronze, Total medal count, and continent. Barcharts were created to see the relationship of medal counts for each continent. The creation of barcharts were visualizations to quickly see how each continent performed in the 2020 Tokyo Olympics.
